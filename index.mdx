Welcome to **Book Paradise** — a premium, fully client-side bookstore experience crafted with HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript. Everything runs in the browser: no server, no backend, no database required. Your cart and wishlist live in `localStorage` so they persist across sessions automatically.

## Getting started

<Steps>
  <Step title="Download the files">
    Grab `index.html`, `style.css`, and `script.js` from the repository.
  </Step>
  <Step title="Open in a browser">
    Double-click `index.html` or serve the folder with any static file server. No build step needed.
  </Step>
  <Step title="Explore the catalog">
    Browse the 24 books, try search and filters, and add a few titles to your cart.
  </Step>
  <Step title="Customize">
    Edit the `books` array in `script.js` and the CSS variables in `style.css` to make it your own.
  </Step>
</Steps>

<Tip>
  Book Paradise uses only CDN-hosted libraries — no `npm install` required. Open `index.html` directly in any modern browser and everything works immediately.
</Tip>
